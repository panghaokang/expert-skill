# Decision Maker — 决策专家摄入

## 补充问题

### Q：决策框架

```
面对一个重要决策时，你的标准思考框架是什么？你用什么方法
把复杂问题拆解成可比较的选项？

例：我习惯用"影响范围 × 实现成本 × 可逆性"三维来评估任何方案。
可逆的决策不需要完美信息，不可逆的决策必须充分讨论。
```

### Q：不确定性处理

```
当你没有足够数据但必须做决定时，你怎么处理？什么情况下
你会推迟决策或向上求助？

例：如果有 70% 的把握就做了，剩下的 30% 靠监控和快速回滚兜底。
但如果涉及资金安全或用户数据，没有 95% 把握我不会动。
```

### Q：经典决策复盘

```
说一个你做过的最艰难（或最成功/最失败）的决策。
回头看，关键的判断依据是什么？学到了什么？
```
